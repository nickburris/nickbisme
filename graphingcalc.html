I created this graphing calculator as a fun JavaScript project. You can input an equation or multiple comma-seperated equations, and it will draw them on the graph. I wrote my own equation interpreter and processor, by implementing the shunting-yard algorithm to convert each equation to reverse polish notation (AKA "postfix"), from which an answer can then be much more efficiently calculated. I also wrote some unit tests for the equation processor.

Cool features include trig functions, proper order of operations, seamless panning, and seamless zooming with the scroll wheel! Check to log latency stats to the console:
